# 這個集成包使用的工具與框架

* Vue3
* Vite
* i18n
* TypeScript
* SASS/SCSS
* CSS Module
* Tailwind CSS
* Element Plus
* Element Plus dark mode
* Vue Router (設定了簡易的 Login / 404 / Home 的路由)
* ESLint

## 執行

```
npm install
npm run dev
```
